# Career Adventure Game

This repository publishes a single-page HTML career adventure game through GitHub Pages.

## Site

The live site is served from `index.html` once GitHub Pages is enabled for the repository.

## Counselor Admin

`admin.html` is the counselor-facing dashboard. It requires the Apps Script backend in
`apps_script_backend_admin_v3.gs` to be installed and configured with the Google Sheet ID
and a counselor registration code.
